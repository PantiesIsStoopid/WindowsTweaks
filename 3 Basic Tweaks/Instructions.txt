System Optimization Steps
Create a Restore Point:

Before making any changes, ensure you create a restore point for safety.
Uninstall Unnecessary Apps:

Go through your installed applications and uninstall any that you don’t need.
Manage Startup Apps:

Open Startup Apps and disable any applications that don’t need to launch on startup.
Use Autoruns:

Open Autoruns and disable unnecessary startup entries, but do not disable:
cmd.exe
Any GPU-related settings.
Disable User Account Control (UAC):

Double-click the UAC setting to disable it.
Run Latency BCD Tweak:

Open the Registry Tweaks and run Latency BCD Tweak as an administrator.
Optimize Windows Settings:

Double-click to optimize all Windows settings.
Adjust Performance Options:

Go to Performance Options and select Adjust for best performance.
Configure Sound Settings:

Open Sound Settings and disable all output devices except for your headphones or speakers.
Adjust Mouse Settings:

In Mouse Settings, uncheck Enhance pointer precision.
Run MarkC Windows Fix:

Double-click to apply MarkC Windows Fix.
Use WinUtil Tweaks:

Open WinUtil, navigate to Tweaks, and select Standard. Set your DNS to Cloudflare.
Apply the Tweaks:

Run the selected tweaks.
Activate Ultimate Profile:

Click Add and activate the Ultimate Profile.
Open System Configuration:

Press Win + R, type msconfig, and hit Enter.
Configure Boot Settings:

Go to the Boot tab:
Check Advanced Options and set the number of processors to the maximum available.
Set the timeout to 10 seconds.
Manage Optional Features:

Open Optional Features and remove all features except for Notepad.
